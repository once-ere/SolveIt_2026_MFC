TRLAN User Guide

Kesheng Wu
Horst Simon

This is edition 1.0 of the user's guide for TRLAN version 1.0. It describes how to use TRLAN to solve real symmetric eigenvalue problems. The main focus is on how to set up the data required by TRLAN so that TRLAN can do its job. It also includes information on installation and debugging.
