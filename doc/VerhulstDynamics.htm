Logistic Map

x {n+1} = R x n ( 1 - x n ), where, 0 <= R <= 4 , 0 <= x n <= 1

The Logistic Map birth rate 'R' is the vertical coordinate.

The Logistic Map population xn is the horizontal coordinate.

Ctrl left-click in the 'System Window' to zoom in.

Ctrl right-click in the 'System Window' to zoom out.


'Environment' »   '# Steps' in the 'Status Window' sets the plot density.
