SolveIt Physics Simulator

The window with the coordinate axes called the 'System Window'.
The window to its left is where commands are typed and is called the 'Command Window'.
The window on the far left is the 'Status Window'.


Help (?)

  • Click the right mouse button when over the Command Window to diplay a menu of examples.
    • Select an example.
    • The commands that define the problem are executed in the Command Window.
  • Do not press 'Enter' to enter a typed command; use 'Shift+Enter'.
  • If the example menus are inactive then press the 'Stop' button to enable them.
    •

Coordinate System:

The red horizontal axis is the x-axis. The green vertical axis is the y-axis. The blue axis normal to the screen is the z-axis.
Graph labeling is crude and will be re-written when time permits.

Command Window:

This simple text user-interface was written quickly and has bugs. If the text in an input or output box disappears then you can scroll it back into view using the arrow keys or mouse.
System Window:

  • Drag with the left mouse button down to rotate the view.
  • control-left-drag an object to move it
  • shift-control-left-drag a vector to rotate it
  • The 'System Window' button toggles the view between perspective projection and parallel projection.
  • The (Page Up, Page Down) keys zoom (in, out) when using parallel projection.
  • The (Left, Right) [respectively, (Up, Down)] arrow keys translate along the x-axis (resp., y-axis) when using parallel projection.
  • Both the screen and system (or world) coordinates of the point under the mouse are printed in the browser status line.
  • The difference between the system coordinates of the point under the mouse and a selected point is also printed in the browser status line.
  • This point is selected by left-clicking on a point while holding down the 'Shift' and 'Ctrl' keys.

Partial Button Summary:
  • press the 'PNG' [portable network graphics] button to save an image in .png format [recommended]
  • press the 'EPS' [encapsulated postscript] button to save an image in .eps format [very slow]
